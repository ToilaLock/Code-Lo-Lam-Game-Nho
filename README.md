# 🎮 Series: Code "Lỏ" Làm Game Nhỏ - Game Creator Club (GCC)

Chào mừng mọi người đã đến với "tổng kho" mã nguồn của series **Code "Lỏ" Làm Game Nhỏ**! Đây là nơi lưu trữ toàn bộ các code từ các video ngắn của series **Code "Lỏ" Làm Game Nhỏ** của **GCC**.
Mục tiêu của series này là biến những thuật toán, logic khô khan thành những tựa game nhỏ vui nhộn, giúp các bạn tân binh dễ dàng tiếp cận với tư duy lập trình game.

## 📂 Danh Sách Các Tập 

| **Tập 1** | Thuật toán Random Walk (Gã say rượu) | C++ (Console) |


## 🛠️ Hướng dẫn cách lấy Code về máy

1. Bấm vào thư mục của Tập bạn muốn lấy code (ví dụ: `Ep01 - RANDOM WALK GAME.cpp`).
2. Mở file `.cpp` bên trong lên.
3. Bấm vào biểu tượng **Copy raw content** (hình hai ô vuông chồng lên nhau ở góc phải) để copy toàn bộ code.
4. Dán vào trình biên dịch của bạn (Visual Studio, Dev-C++,...) và bấm **Run** để trải nghiệm!
